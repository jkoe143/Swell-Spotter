# Swell Spotter

A web application for visualizing ocean wave conditions and generating safe maritime routes.

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Open `http://localhost:5173` in your browser
2. View wave conditions on the map
3. Use time controls to navigate historical data
4. Select ports and click "Generate Route" for maritime routing
